El certificado ISO consta de un conjunto de normas elaboradas por la Organización Internacional de Estandarización (ISO) con el propósito de ordenar la gestión dentro de las empresas en sus diferentes ámbitos y departamentos.

Tipos de certificados ISO y características

F.O.P.S ISO 3449

  • Específica pruebas de laboratorio para medir las características estructurales, y proporciona requisitos de rendimiento en una prueba representativa para estructuras de protección contra objetos que caen (FOPS) destinadas a su uso en máquinas de movimiento de tierra montadas según se define en ISO 6165.

R.O.P.S ISO 3471

  • Especifica los requisitos de rendimiento para las estructuras de protección antivuelco metálicas (ROPS) para maquinaria de movimiento de tierra, así como un medio consistente y reproducible de evaluar el cumplimiento de estos requisitos mediante pruebas de laboratorio utilizando carga estática en una muestra representativa.

L.T.D.A. ISO 15818

  • Especifica los requisitos de rendimiento para los puntos de fijación de elevación y amarre de maquinaria de movimiento de tierras como se define en ISO 6165.

ISO 15818

  • 2017 también se aplica a los componentes y subconjuntos de maquinaria de movimiento de tierras que el fabricante pretende levantar o atar por separado mediante el uso de puntos de fijación de elevación o amarre.
  • Elevación con grúas (por ejemplo, grúas móviles, grúas de pórtico)
  • Transporte por carretera (por ejemplo, camión, remolque)
  • Tnonransporte ferroviario, incluido el transporte combinado (por ejemplo, vagones con contenedores, cajas móviles, semirremolques, camiones).
